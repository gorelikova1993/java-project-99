# Task Manager

## Описание

Task Manager – это система управления задачами, аналогичная [Redmine](http://www.redmine.org/).  
Она позволяет:
- Ставить задачи
- Назначать исполнителей
- Менять статусы задач

Для работы с системой требуется регистрация и аутентификация.
